## Installation

```bash
npm create vite@latest
```
```bash
cd project
```
```bash
npm i
```

```bash
npm install tailwindcss @tailwindcss/vite
```

### vite configuration
```bash
import tailwindcss from '@tailwindcss/vite'
```
....
```bash
 plugins: [
    tailwindcss(),
  ],
```


### CSS
```bash
@import "tailwindcss";
```

### Start
```bash
npm run dev
```
